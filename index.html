The basic of Web & HTML

Main Terminology
1- HTML - Hypertext Markup Language
2- WWW - World wide web
3- How the web (Internet) works: The web is a bunch of computers that communicate with each other.
When a person goes to a web page like www.google.com, their computer sends a HTTP Request to a server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.
Why Computers are Stupid: Computers are stupid because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause huge problems in a program.

Key Points of HTML

1- HTML - main type of doc on the web
2- Pain text, HTML, Images, Videos and Music - all these types of files can be found on web.
3-HTML is made up of;
a- Text Content - what you see
b-Markup - what it looks like ref to other documents (example images & videos)
c- Links to other pages.
4- HTML documents are made of HTML Tags and Elements.
When writing HTML, we tell browsers the type of each element by using HTML tags. This video explains the distinction well.
5- HTML elements are either inline or block.
Examples of Inline elements are; (all in angle brackets) a, br, span, img, strong, em.
Block elements form an "invisible box" around the content inside of them.
Examples of block elements are; (all in angle brackets) div, p, form.
6- Attributes: This is a property of an HTML element. For example; to set the "href" attribute of anchor tag to the URL.
Void Tag: Doesn't require a closing tag For example; "br" tag.
